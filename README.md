# 🌐 WebVerse

🚀 **AI-Powered Website Builder** built with **React**, **Next.js**, and **Google Gemini**, enabling users to generate fully responsive websites from natural language prompts.

[🔗 Live Demo on Render](https://webverse-x506.onrender.com/)

---

## ✨ Features

- 🧠 **AI-Powered Generation**: Generates complete websites using **Google Gemini** from simple natural language input.
- 🎨 **Modern UI/UX**: Built with **ShadCN** and **TailwindCSS** for a clean, accessible, and fully responsive interface.
- 🔒 **Google OAuth Authentication**: Secure and easy sign-in via Google.
- 🧪 **Live Preview Sandbox**: Allows users to interact with generated websites in real-time.
- 🗃️ **MongoDB Storage**: Stores generated websites and user interaction history for persistence and reusability.

---

## 🛠️ Tech Stack

- **Frontend**: React, Next.js, TailwindCSS, ShadCN
- **AI Integration**: Google Gemini API
- **Authentication**: Google OAuth via NextAuth.js
- **Database**: MongoDB (Atlas)
- **Deployment**: [Render](https://webverse-x506.onrender.com/)
